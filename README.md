## 📂 Featured Projects

### 🛒 [Predicting Product Sales via Sentiment Analysis](https://github.com/ketan3107/Thesis-)
> **MSc Thesis Project | 2024**

Built an end-to-end ML pipeline to predict Amazon product sales from customer review sentiment — addressing the real-world business problem of demand forecasting without historical sales data.

- Engineered a custom sales proxy metric using review rating counts
- Applied **NLP techniques**: Sentiment Analysis + Word2Vec embeddings
- **XGBoost** achieved R² = 0.55, outperforming classification baselines (57% accuracy)
- Business insight: High-sentiment products identified as high-potential SKUs for proactive stock and marketing decisions
- **Stack:** Python • XGBoost • NLP • Word2Vec • Scikit-learn • Pandas • EDA
